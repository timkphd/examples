Some Examples

Here are some not complelety trivial makefile examples. In The "C" directory we have a single example and in the "fort" directory we have two examples. In the simple directory we have three additional makefiles that build Fortran and C programs

See the comments in the makefiles for additional information.

Warning:

Makefiles contains "tab" characters and they will not work properly without them. If you copy/paste a makefile the tab might get converted to spaces.

Download makeExamples.tgz to get all of the example files. You can use one of the following two commands to do the download.

  • wget http://geco.mines.edu/files/userguides/scripting/make/examples/makeExamples.tgz
  • curl http://geco.mines.edu/files/userguides/scripting/make/examples/makeExamples.tgz > makeExamples.tgz
Then do a:
  • tar -xzf makeExamples.tgz

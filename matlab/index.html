Matlab examples

This directory contains several Matlab examples that have been tested on Summit. To run the examples on Summit in your "projects" directory copy/paste the following commands: 


# Go to your projects directory
cd /projects/$USER
# Create a new directory for your runs and go there
mkdir mtest
cd mtest
# Download the examples as a *tgz file
wget -q petra.acns.colostate.edu/docs/matlab/matlab.tgz
# Uncompress the file
tar -xzf matlab.tgz
# Go to the "canned" directory
cd canned/
# Submit the job
sbatch run_matlab
# Go to the "invert" directory
cd ../invert/
# Submit the job
sbatch run_matlab
# Go to the "stom2" directory
cd ../stom2/
# Submit the job
sbatch run_matlab

The README files have descriptions of the examples.
